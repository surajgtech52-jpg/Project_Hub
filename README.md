# Campus Dine 🍔

A full-stack canteen management and hyperlocal food ordering platform designed for campus ecosystems.

## 🚀 Key Features
* **Role-Based Access**: Separate interfaces for Students, Teachers, and Canteen Administrators.
* **Live Order Tracking**: Real-time status updates using AJAX (Preparing, Ready, Collected).
* **Smart Menu**: Admins can toggle "Sold Out" items and schedule specific category availability times.
* **Receipt System**: Automated digital receipts with a "Full-Screen ID" view for quick counter pickup.

## 🛠️ Tech Stack
* **Language**: PHP
* **Database**: MySQL
* **Frontend**: Bootstrap 5, jQuery, FontAwesome
* **Real-time**: AJAX (Asynchronous JavaScript and XML)

## ⚙️ Setup
1. Import the SQL file from the `/sql` folder into your MySQL database.
2. Update the database connection settings in `db_connect.php`.
3. Host the files on a PHP-enabled server (XAMPP, WAMP, or any web host).
